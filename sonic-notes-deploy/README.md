## ☁️ Deploy to Vercel

### Option 1 — Drag & Drop (Easiest)
1. Go to vercel.com → Log in
2. Click "Add New Project"
3. Drag and drop this folder
4. Done! Live in 30 seconds.

### Option 2 — Via GitHub
1. Push this folder to: github.com/madansp7676/sonic-notes
2. Go to vercel.com → Import Git Repo
3. Select the repo → Deploy

### Option 3 — CLI
```bash
npm install -g vercel
vercel login
vercel --prod
```
